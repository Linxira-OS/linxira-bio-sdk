# VCF 变异统计

## 用途

对 VCF 记录、ALT 等位基因、FILTER、染色体、样本和基因型缺失情况进行确定性
描述统计。

## 输入

一个带有有效 `##fileformat` 声明和 `#CHROM` 表头的 VCF 文件。程序按内容识别
纯文本、gzip 和 BGZF；不接受 BCF。

## 参数

输入路径为必需参数。v1 没有可调科学参数；`--json` 返回标准分析结果封装。

## 输出

返回记录数与样本数、PASS 与命名过滤记录数、SNP/indel/MNV/symbolic ALT 等位
基因数、多等位记录数、转换/颠换数及比值、已检出与缺失基因型数及缺失率、各
染色体记录数和警告。

## 示例

```bash
linxira-bio variant stats tests/fixtures/variant-stats/mixed.vcf --json
```

## 结果解读

记录指标按行计数，变异类别指标按 ALT 等位基因计数；一条多等位记录可增加多个
类别。FILTER `.` 既不是 PASS，也不是命名过滤值。Ti/Tv 仅统计双等位单碱基
替换。含任一缺失等位的基因型视为缺失；没有 `GT` 字段的记录不进入缺失率分母。

## 注意事项

该能力不做等位基因规范化、参考基因组版本比较、深度或似然评估、变异注释、
致病性判断或样本身份验证。不同数据集必须经过等价的预处理和过滤后才能比较。

## 运行时依赖

纯 Rust 本地流式能力，无需 Python、R、Java、htslib 或外部命令行工具。

## 引用

字段语义遵循 GA4GH VCF 规范。当前解析器面向 VCF 4.x 文本记录，并会报告不支持
或损坏的结构。

## 故障排除

表头、记录、FORMAT 或基因型等位索引损坏时，按错误中的行号定位。BCF 请先用
bcftools 等成熟工具转换为 VCF。
